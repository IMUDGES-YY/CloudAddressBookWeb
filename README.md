# 服务器接口
## 返回码含义:
| 返回码        | 含义   |
| :----: | :----:  |
|  0     | 登陆成功  |
| -1     | 处理参数   |
| -2     | 处理超时   |
| -3     | 用户名或密码错误   |
| -4     | 登录状态无效   |
| -5     | 短信验证码错误或者已经失效|
| -6     | 用户已存在|
| -7     | 发送验证码太频繁|
